JavaScript Lisp

This is a simple implemenation of Lisp in JavaScript. This was a "blind go" at implementing a Lisp runtime system without looking at existing implementations. This implemenation contains the following built in:

    Macros:
  • def
  • if
  • set
  • lambda
  • let
    • Functions:
  • do
  • eval
    • List Functions:
  • head
  • tail
  • last
  • front
    • Operator Functions:
  • +
  • -
  • *
  • /
    • Equality Functions:
  • =
  • >
  • <
  • >=
  • <=
    • Logical Functions:
  • or
  • and

Example Code:

This will alert the first 10 fibinocci numbers

(def fib (lambda (x)
  (if (<= x 2)
    1
    (+ (fib (- x 1)) (fib (- x 2))))))

(def fibToTen (lambda (start)
  (if (<= start 10)
    (do
      (alert (fib start))
      (fibToTen (+ start 1)))
    "done")))

(fibToTen 1)
